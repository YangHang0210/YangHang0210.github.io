Bookmarklet 方式（推荐，零安装）

一次性设置：把下面的按钮拖拽到浏览器书签栏
Tunnel2cURL

详情页用法：
1. 在 Chrome 中打开任意 Tunnel 请求详情页
2. 点击书签栏上的「Tunnel2cURL」
3. cURL 命令会自动复制到剪贴板，同时弹窗显示

列表页用法（新）：
1. 在 Tunnel 请求列表页点击「Tunnel2cURL」，激活列表模式（顶部会出现蓝紫色横幅）
2. 点击任意请求行，自动获取该请求详情并生成 cURL，复制到剪贴板
3. 再次点击「Tunnel2cURL」或点击横幅上的「关闭」可退出列表模式

原理：Bookmarklet 在 Tunnel 页面上下文中执行，可以直接访问 Tunnel 的 API 和 Cookie，无跨域问题。列表页模式通过 Vue 组件状态和 Tunnel API 自动获取请求详情。
