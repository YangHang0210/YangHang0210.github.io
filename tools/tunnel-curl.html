Bookmarklet 方式（推荐，零安装）

一次性设置：把下面的按钮拖拽到浏览器书签栏
Tunnel2cURL

使用方法：
1. 在 Chrome 中打开任意 Tunnel 请求详情页
2. 点击书签栏上的「Tunnel2cURL」
3. cURL 命令会自动复制到剪贴板，同时弹窗显示

原理：Bookmarklet 在 Tunnel 页面上下文中执行，可以直接访问 Tunnel 的 API 和 Cookie，无跨域问题。
